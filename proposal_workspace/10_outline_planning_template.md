# 10. 목차·배점 연계 계획 템플릿

## 목적
요구사항/배점/section type을 연결한 실행 가능한 목차 계획을 만든다.

## 목차 계획표
| 절 번호 | 절 제목 | section_type(주/보조) | 연결 요구사항 ID | 평가항목/배점 | 목표 분량(A4) | 작성 담당 | 검토 담당 | 비고 |
|---|---|---|---|---|---|---|---|---|
| 1 |  |  |  |  | 0.5~1 |  |  |  |
| 2 |  |  |  |  | 0.5~1 |  |  |  |

## 점검 체크리스트
- [ ] 고배점 평가항목이 독립 절로 배치되었는가?
- [ ] 절별 목표 분량이 과도하지 않은가?
- [ ] section_type이 절 목적과 일치하는가?
- [ ] 작성/검토 담당자 배정이 완료되었는가?
