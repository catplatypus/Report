# final 폴더 사용법

## 목적
검토 완료된 절별 초안을 병합하여 제출용 최종본을 관리한다.

## 운영 방식
1. `drafts/`에서 REVIEW 완료 문서만 선별한다.
2. 목차 순서대로 병합해 단일 제출본을 만든다.
3. 용어, 문체, 표 번호, 참조 번호를 최종 정리한다.

## 최종 점검 체크리스트
- [ ] 모든 요구사항 ID가 최소 1회 이상 반영됨
- [ ] 평가항목 고배점 파트가 충분히 서술됨
- [ ] 과장 표현/금지 문형 미사용
- [ ] 절별 분량 및 전체 분량 기준 충족
- [ ] 오탈자 및 형식 일관성 확인
