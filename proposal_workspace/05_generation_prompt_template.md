# 05. 절별 초안 생성 프롬프트 템플릿

## 사용 목적
section card 정보를 기반으로, 공공기관 용역 제안서의 특정 절을 A4 0.5~1쪽으로 생성한다.

## 입력 패키지(카드에서 복사)
- 절 제목:
- section_type:
- 작성 목적:
- 연결 요구사항 ID:
- 요구사항 원문 요약:
- 평가항목/배점:
- 필수 포함 항목:
- 입력 근거(회사 프로필/기존 실적/정책·법규 등):
- 반드시 포함할 수치/고유명사:
- 표/본문 형식 지시:
- 금지 표현:

## 프롬프트 본문
아래 입력을 기반으로 [절 제목] 초안을 작성하라.

[출력 제약]
1) 한국어로 작성한다.
2) 공공기관 평가자가 읽는 문체로 작성한다.
3) 분량은 A4 0.5~1쪽 수준으로 제한한다.
4) 과장 표현, "단순 ~이 아니라" 문형을 사용하지 않는다.
5) 요구사항과 직접 관련된 내용만 작성한다.
6) 필요한 경우 소제목 2~4개로 구성한다.
7) 표를 넣는 경우 표 아래에 해설 문단을 붙인다.
8) 근거 없는 확정 표현(예: 반드시 달성) 사용을 피한다.

[출력 형식]
- 제목:
- 본문:
- (선택) 표:
- 요구사항 반영 근거 요약(3~5줄):
- 미반영/추가확인 필요 항목(없으면 "없음"):
- 자기점검 체크리스트:
  - [ ] 요구사항 반영
  - [ ] 분량 준수
  - [ ] 금지 표현 미사용
  - [ ] 검증 가능한 표현 사용
