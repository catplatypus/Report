# PROJECT_UNDERSTANDING 전용 필드 템플릿

## 목적
발주기관의 사업 배경, 현황, 문제정의, 기대효과를 정확히 이해했음을 입증한다.

## 입력정보
- 사업 추진 배경/정책 문맥
- 현행 업무·시스템 현황
- 문제점 및 개선 필요사항
- 발주기관 목표/성과지표

## 필수 포함 항목
- 사업 목적 재정의(기관 관점)
- 현황 진단 요약
- 핵심 이슈 3개 내외
- 제안 범위와 목표의 정합성

## 표/본문 형식
- 권장 표: "현황-문제-개선방향" 3열 표 1개
- 본문: 문제정의 → 목표정렬 → 기대효과 순

## 검토 기준
- 기관 맥락을 오해 없이 반영했는가?
- 문제정의가 추상적이지 않고 근거가 있는가?
- 뒤 절(전략/방법)과 자연스럽게 연결되는가?
