# section_cards 폴더 사용법

## 목적
절별 작성 단위를 카드로 관리하여, 담당자 분업과 품질관리를 동시에 수행한다.

## 사용 절차
1. `04_section_card_template.md`를 복사해 실제 카드 파일을 만든다.
2. 절의 성격에 맞는 `*_template.md`를 열어 전용 필드를 채운다.
3. 공통 필드(요구사항 ID, 분량, 완료기준, 상태)를 먼저 확정한다.
4. 카드 기준으로 초안을 생성하여 `drafts/`에 저장한다.

## GPT 입력 품질 기준
- 요구사항 ID와 요구사항 원문 요약을 반드시 함께 적는다.
- 평가항목/배점, 필수 수치, 금지 표현을 누락하지 않는다.
- 카드만 전달해도 초안 생성이 가능하도록 입력자료를 구체화한다.

## 파일 네이밍 권장
- `SC-01_PROJECT_UNDERSTANDING.md`
- `SC-02_STRATEGY.md`

## 운영 규칙
- 카드 1개 = 원칙적으로 제안서 절 1개
- 카드 상태는 TODO → DOING → REVIEW → DONE 순으로 변경
- REVIEW 단계에서 요구사항 누락이 발견되면 DOING으로 되돌린다.
