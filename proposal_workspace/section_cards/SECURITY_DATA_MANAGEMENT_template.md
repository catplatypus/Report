# SECURITY_DATA_MANAGEMENT 전용 필드 템플릿

## 목적
보안·개인정보·데이터관리 요구사항을 충족하는 통제체계를 제시한다.

## 입력정보
- 보안 관련 법규/지침
- 개인정보 처리 범위
- 접근권한/암호화/백업 정책
- 사고 대응 체계

## 필수 포함 항목
- 보안 정책 및 책임체계
- 데이터 수집·저장·폐기 원칙
- 접근통제/로그관리/암호화 방안
- 보안사고 대응·보고 절차

## 표/본문 형식
- 권장 표: "위협-통제방안-점검주기-책임자" 표
- 본문: 준수체계와 운영 절차 설명

## 검토 기준
- 법규 및 발주기관 지침 반영 여부
- 보안 통제가 실무 수준으로 구체적인가?
- 사고 대응 절차가 누락되지 않았는가?
